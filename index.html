Apa itu PPLG?

PPLG (Pengembangan Perangkat Lunak dan Gim) adalah jurusan yang mempelajari cara merancang, membuat, dan mengembangkan perangkat lunak (software) serta gim (game). Jurusan ini membekali siswa dengan keterampilan coding, desain, analisis sistem, serta pengujian aplikasi agar siap bersaing di dunia kerja maupun melanjutkan studi.

Struktur Kurikulum

Mata Pelajaran Keterangan
Pemrograman Dasar Belajar logika & dasar coding
Pemrograman Web & Mobile Membangun aplikasi website & Android/iOS
Basis Data Mengelola data menggunakan MySQL / MongoDB
Pengembangan Gim Membuat game dengan Unity, Godot, atau Construct
Desain UI/UX Merancang tampilan aplikasi yang menarik

Visi & Misi

Visi

Menjadi jurusan unggulan yang menghasilkan lulusan kreatif, inovatif, dan berdaya saing tinggi di bidang teknologi informasi dan gim.

Misi

  • Meningkatkan keterampilan pemrograman siswa.
  • Mengembangkan budaya kerja kreatif & profesional.
  • Menghasilkan karya nyata berupa aplikasi & gim.
  • Menjalin kerjasama dengan industri teknologi.

Kenapa Memilih PPLG?

Prospek Kerja

Software Engineer, Game Developer, Web Developer, Mobile Developer, UI/UX Designer.

Keunggulan

Didukung guru profesional, materi up-to-date, serta proyek nyata sesuai industri.

Tujuan

Mencetak lulusan yang siap kerja di bidang teknologi, kreatif, dan inovatif.

Hasil Project Siswa

Aplikasi Web Sekolah

Website sekolah dengan fitur informasi siswa, guru, dan berita.

Game Edukasi

Game interaktif berbasis Unity yang mendidik sekaligus menghibur.

Aplikasi Mobile

Aplikasi Android untuk manajemen keuangan sederhana.
